# Bank-analysis-project-
This project focuses on analyzing banking customer data to uncover insights from customer demographics, transactions, and credit profiles. It includes data extraction from a MySQL database, data cleaning (handling null values and outliers), statistical analysis, and visualizations to better understand customer behavior and financial patterns.
